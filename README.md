os_monasca
==========

Install and configure OpenStack Monasca.

The following services are installed:
- Chrony
- MariaDB
- Zookeeper
- Kafka
- Storm
- InfluxDB
- Grafana
- monasca-api
- monasca-persister
- monasca-notification
- monasca-thresh

Requirements
------------

none

Example Playbook
----------------

      hosts: monasca
      user: root
      roles:
        - { role: "os_monasca", tags: [ "os-monasca" ] }

        - keystone_ip_address: "{{ groups['devstack'][0] }}"
